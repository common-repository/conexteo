== Description ==

Our module allows you to easily send SMS to your customers to inform them of their order, a change in status or any other important message.
Integrated contact synchronization.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the dir to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Set up your api key and secret key in the settings page
